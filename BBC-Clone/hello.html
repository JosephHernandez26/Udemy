Positioning

Relative positioning

set the position to relative: positions relative to where it would normally be. add "left" or "right" and add a px of distance (including negative numbers)

to move up and down - add the "top" property

note that "top" basically adds a margin-like gap to the top, essentially moving the box down

The Z index

to change elements from being in front of another to behind or visa versa, set the z index.

the element with the highest z index will be positioned in the front.

Opacity

how opaque an element appears (how see-through it is)

the bigger the number the more see-through it is.

Absolute positioning

removes the element out of the flow of the page and positions it in a default position (top left).

this positions the element over the top of everthign else in the page

Fixed positioning

the element stays in position regardless of scrolling behavior

Margins

the margin encircles the elements

Specific Margins

you can make each side of the box to different margins as desired (top, right, bottom, left)

to set an individual margin, select margin-"whichever side".

Padding

works on the inside of an element rather than the outside (margins=outisde).

provides spacing within the element, between the margin and the content.

works identically to margins (top, right, bottom, left)

when a width of pixels is set, padding ADDS to the overall pixel count.

pro-tip

set the margin and the padding for the body to - 0 before adding styling.

use a CSS reset to make sure your styling translates to various browsers.

Borders

can be added in one line (size, color, and type)

you can alter each border and they flow in the same way as the others (top, right, bottom, left).

Border Radius

adds a radius to the border.

border radius of 50% makes the box a circle.

Fonts

Font-Family

the type of font to be used (times, san serif, arial, etc.)

if you see three font names, the subsequent font names are the "fallback" fonts should a browser not support the initially chosen one.

look up websafe fonts and google fonts.

Google fonts

simply cut and paste the link from the selected font and place within a link tag in the head.

replace the font family with the provided code.

Styling Text

Bold Text

set "font-weight" to bold.

Italic

Set the "font-style" to italic.

to make a line of text BOTH bold and italic set a "span" class and set the p-class to italic.

Underlined

set "text-decoration" to underlined.

No stylling

to remove underlining for links set "a" and "text-decoration" to none.

:hover

this pseudo selector will style the link when the mouse hovers over it.

Text Size

set "font-size" to whatever size you choose ex: 200% = twice the normal size.

Text Align

text align will align the text to the specified side of the PARENT element.

text align is set to the left by default

Justified

stretches the text all the way to the edge of the page

Lorem Ipsum

use this to populate areas of the webpage with random text that appears to be naturally written.

Styling Links

Opening in a new tab

set the "target" attribute of the link to "_blank".

:Vistied

changes the color of the links that have been clicked to a different color by applying the pseudo class of "":visited"
